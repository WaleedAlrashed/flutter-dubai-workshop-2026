## About This Workshop

> "In 2026, knowing *how* to write a for-loop is table stakes. Knowing *when*, *why*, and *how to architect* systems is what separates effective developers from prompt-typists."

This repository contains all demo projects, capstone increments, and teaching materials for the **Flutter for the Agentic Era** workshop. The course teaches Flutter through the lens of **AI-augmented development** — using tools like Claude Code, Cursor, and Copilot throughout, while focusing on architecture decisions AI can't make for you.

## Workshop Structure

| Day | Theme | Projects | Key Concepts |
|-----|-------|----------|-------------|
| [**Day 1**](./day1-flutter-foundations/) | Flutter Foundations | 7 mini demos | Widgets, layout, theming, Gemini AI |
| [**Day 2**](./day2-forms-state-apis/) | Forms, State & APIs | Volunteer Connect v2 | Forms, validation, Riverpod, mock APIs |
| [**Day 3**](./day3-navigation-architecture/) | Navigation & Architecture | Volunteer Connect v3 | GoRouter, Dio, feature-based architecture |
| [**Day 4**](./day4-production-patterns/) | Production Patterns | Volunteer Connect v4 | Clean Architecture, Result pattern, interceptors |
| [**Day 5**](./day5-firebase-deployment/) | Firebase & Deployment | Volunteer Connect v5 | Auth, testing, localization, Cloudflare Pages |

## Capstone Project: Dubai Volunteer Connect

A volunteer management app inspired by the [Hamdan Volunteer Platform](https://app.hamdanvolunteer.ae/), built incrementally across 5 days:

```
Day 1: Widget foundations & Dubai-themed UI
Day 2: + Forms, validation, Riverpod state management
Day 3: + GoRouter navigation, Dio networking, feature architecture
Day 4: + Result pattern, interceptors, Gemini AI chat, rank badges
Day 5: + Auth flows, testing, localization, deployment
```

## Tech Stack

- **Flutter** 3.41.4 / **Dart** 3.11.1
- **State Management:** Riverpod
- **Navigation:** GoRouter
- **Networking:** Dio
- **AI:** Google Generative AI (Gemini 2.5 Flash)
- **Theming:** Material 3 with `ColorScheme.fromSeed`
- **Deployment:** Cloudflare Pages
